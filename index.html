About

Texar is a highly modularized, versatile, and extensible toolkit designed to support the broad set of machine learning applications, especially text generation such as machine translation, dialog, summarization, text manipulation, etc. Texar extacts the common patterns underlying the diverse tasks and methodologies, creates a library of highly reusable modules and functionalities, and enables arbitrary model architectures and algorithmic paradigms like maximum likelihood learning, adversarial learning, reinforcement learning, etc. Texar emphasizes heavily on extensibility and modularized system design, so that components can be freely plugged in or swapped out.
